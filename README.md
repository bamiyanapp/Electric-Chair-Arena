## プロジェクト概要

日本企業に存在する「形式主義」「ハンコ文化」「マナー文化」を風刺したジョークWebアプリ。

プレイヤーは申請書への捺印を通じて承認を得ようとするが、次々と現れる謎マナーや独自ルールによって差し戻される。

最終的に最高位の捺印作法を習得した瞬間、電子承認システム導入により全てが無意味になる。

# 世界観

捺印作法が国家資格となった近未来日本。

企業では業務能力よりも捺印マナーが重視されている。

プレイヤーは「ハンコマスター」の称号を目指す。
