# Electric-Chair-Arena

電気イスゲームのルールを実装し、戦績データを蓄積しながらAIプレイヤーの育成・対戦・ランキング評価を行う研究プロジェクト。

---

# プロジェクト情報

## 概要
日本企業に存在する「形式主義」「ハンコ文化」「マナー文化」を風刺した要素（旧：ハンコマスター検定）をベースにしつつ、戦略的カードゲーム「電気イスゲーム」をコアとした対戦・研究プラットフォーム。

---

# 開発目的

## 主目的
* 戦略ゲームにおけるAIの意思決定アルゴリズムの研究
* Next.js / Serverless 構成によるモダンな Web アプリケーション開発の習得

## 学習テーマ
* Webアプリ開発 (Next.js, React, TypeScript)
* AI活用 (意思決定ロジック, 強化学習的アプローチ)
* ゲーミフィケーション
* UI/UX (直感的なカード操作)

---

# システム構成

## Frontend
* Next.js (App Router)
* React
* TypeScript
* Tailwind CSS

## Backend
* AWS Lambda (Serverless Framework)
* Node.js
* Vitest (Testing)

## Database
* DynamoDB (予定) / SQLite (開発用)
